# English Words Flash Card

A beautiful Qt QuickQML application for learning English words with flashcard functionality.

![Flashcard App](screenshot.png)

## Features

- Dark Theme Modern dark UI with elegant color scheme
- 3D Flip Animation Smooth card flip effect
- SQLite Database Stores English words with definitions
- Random Word Selection Learn words in random order
- Keyboard Shortcuts Space to flip, arrow keys for navigation
- Responsive Design Adapts to different screen sizes

## Requirements

- Qt 6.5 or higher
- CMake 3.16 or higher
- C++17 compatible compiler
- SQLite3
